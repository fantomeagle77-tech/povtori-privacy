Политика конфиденциальности приложения «Повтори»

Дата обновления: 30.03.2026

Коротко: «Повтори» — локальное приложение-память. Основные данные пользователя хранятся на его устройстве. Приложение не требует создания аккаунта и не продаёт пользовательские данные.

1. Какие данные может хранить приложение

Приложение позволяет пользователю создавать и хранить карточки сценариев, фотографии, голосовые заметки, историю выполнений, локальный PIN-код и резервные копии, которые создаются самим пользователем.

2. Где хранятся данные

Основные данные приложения хранятся локально на устройстве пользователя. Это может включать локальную базу данных приложения, прикреплённые изображения, голосовые заметки и экспортированные резервные копии.

3. Передача данных

Приложение не предназначено для продажи пользовательских данных и не передаёт содержимое карточек, заметок, фотографий или истории третьим лицам, кроме случаев, когда пользователь сам выполняет экспорт, копирование или передачу данных через функции устройства.

4. Разрешения устройства

  • доступ к фото или файлам — для выбора изображений;
  • доступ к микрофону — для записи голосовых заметок;
  • доступ к уведомлениям — для локальных напоминаний, если эта функция включена;
  • доступ к файловой системе — для экспорта и импорта резервных копий.

5. Безопасность

Приложение поддерживает локальную защиту PIN-кодом. Пользователь сам отвечает за сохранность устройства, экспортированных резервных копий и локальных файлов.

6. Удаление данных

Пользователь может удалить карточки, вложения и приложение со своего устройства. Если резервная копия заранее не создана, локальные данные после удаления приложения могут быть утрачены.
